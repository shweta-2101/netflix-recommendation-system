# Netflix Recommendation System

A personalized movie recommendation system built on the Netflix Prize Dataset
using Collaborative Filtering and SVD Matrix Factorization.

## Team

## Project Overview
This project builds an intelligent recommendation engine capable of:
- Learning user preferences from historical interactions
- Predicting ratings for unseen content
- Generating personalized Top-10 recommendations
- Comparing traditional and advanced recommendation approaches

## Dataset
Netflix Prize Dataset from Kaggle:
https://www.kaggle.com/datasets/netflix-inc/netflix-prize-data

### Dataset Statistics
- Total ratings: 100,480,507
- Users: 480,189
- Movies: 17,770
- Rating scale: 1-5 stars

### Sampled Subset Used
- Total ratings: 18,829,034
- Users: 176,959
- Movies: 17,767
- Sparsity: 99.40%

## Models Implemented
### Model 1 — Item-Based Collaborative Filtering
- Computes cosine similarity between movies
- k=30 nearest neighbors
- RMSE: 1.1195 | MAE: 0.8630 | MAP@10: 0.9376

### Model 2 — SVD (Matrix Factorization)
- Learns latent factors from user-movie interactions
- n_factors=100, n_epochs=20
- RMSE: 1.0373 | MAE: 0.8252 | MAP@10: 0.9383

## Results
| Model | RMSE | MAE | MAP@10 |
|-------|------|-----|--------|
| Item-Based CF | 1.1195 | 0.8630 | 0.9376 |
| SVD | 1.0373 | 0.8252 | 0.9383 |

**Winner: SVD on all three metrics**

## Key Findings
- SVD outperforms Item-Based CF on RMSE, MAE and MAP@10
- Both models achieve MAP@10 > 0.93 showing strong ranking quality
- 67.2% of SVD predictions within 1.0 star of actual rating
- Item-Based CF struggles with 99.4% sparsity — motivates SVD
- Both models agree on recommendations for same user validating quality

## How to Reproduce Results

### Requirements
pip install pandas numpy matplotlib seaborn scikit-surprise jupyter

### Steps
1. Download Netflix Prize dataset from Kaggle link above
2. Place raw files in `data/raw/` folder
3. Run notebooks in order:
01_data_loading_and_sampling.ipynb  → processes raw data
02_eda.ipynb                         → exploratory analysis
03_model_collaborative_filtering.ipynb → trains Item-Based CF
04_model_svd.ipynb                   → trains SVD model
05_evaluation_and_comparison.ipynb   → RMSE + MAP@10 evaluation
06_recommendations.ipynb             → final recommendations

### Notes
- Raw data files (~20GB) are not included in repository
- Processed CSV files are generated by notebook 01
- Models are retrained by running notebooks 03 and 04
- random_state=42 used throughout for reproducibility
- Recommended to run on Google Colab for sufficient RAM

## Evaluation Metrics
- **RMSE** — measures rating prediction accuracy (lower is better)
- **MAE** — measures average prediction error (lower is better)  
- **MAP@10** — measures ranking quality of top 10 recommendations (higher is better)
- Relevance threshold: actual rating >= 3.5 (as per problem statement)
